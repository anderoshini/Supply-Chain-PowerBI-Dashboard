# Supply-Chain-PowerBI-Dashboard

## Project Overview
This Power BI dashboard provides end-to-end visibility into supply chain operations through CEO, CFO, and Manufacturing dashboards.

## Dashboards

### CEO Dashboard
- Revenue Analysis
- Units Sold
- Profit Margin
- Stock Level Monitoring
- Top Revenue Generating SKUs
- Customer Demographics Analysis

### CFO Dashboard
- Total Profit
- Revenue and Cost Analysis
- Manufacturing Cost by Category
- Transportation Cost Analysis
- Top Expensive SKUs

### Manufacturing Dashboard
- Production Volume Tracking
- Defect Rate Analysis
- Lead Time Monitoring
- Inventory Management
- Supplier Performance Analysis

## Tools Used
- Power BI
- Microsoft Excel
- DAX
- Data Modeling
- Data Visualization

## Key Features
- Interactive Slicers
- KPI Cards
- Geographic Analysis
- Executive-Level Reporting
- Supply Chain Performance Monitoring

## Dataset
Supply Chain Operations Dataset
